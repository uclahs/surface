# Breadcrumb

The Breadcrumb component provides a visual representation of a page's path.

## Breadcrumb technical specs

* `breadcrumb[]` It's an array of items each of which represents a hierarchy level in a page's path.
  * `url` it's the individual link's url
  * `text` it's the individual link's text or label
