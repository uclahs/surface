# Eyebrow

The Eyebrow element usually prints a string of text that typically represents a category for the given content.

## Eyebrow technical specs

* `eyebrow[]` is an object with the following properties:
  * `modifier` to pass a CSS class.
  * `text` is the actual text string for the label.
  * `url` if available, the label becomes a link.
